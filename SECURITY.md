# Security Policy

## Supported Versions

The latest released version is supported.

| Version | Supported |
| ------- | --------- |
| latest  | :white_check_mark: |
| older   | :x: |

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Use one of these private channels:

1. GitHub Security Advisories (preferred):
   - https://github.com/Feberdin/NextFirst/security/advisories/new
2. If advisory creation is not possible, open a private communication with the maintainer and include:
   - affected version
   - reproduction steps
   - impact assessment
   - suggested mitigation (if known)

## Response Targets

- Initial acknowledgment: within 72 hours
- Triage decision: within 7 days
- Patch timeline: depends on severity and complexity

## Disclosure Policy

- We coordinate responsible disclosure.
- A fix is prepared before public disclosure whenever reasonably possible.
